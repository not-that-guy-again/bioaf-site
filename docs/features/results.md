bioAF provides multiple ways to explore and share your analysis results — from interactive single-cell viewers to publication-ready QC reports.

## cellxgene

For single-cell datasets, bioAF integrates [cellxgene](https://chanzuckerberg.github.io/cellxgene/) — an interactive explorer that lets you:

- Visualize UMAP and t-SNE embeddings
- Color cells by gene expression, cluster assignment, or sample metadata
- Select and compare cell populations
- Export visualizations

<img src="{{ '/assets/images/screenshot-plot-preview.png' | relative_url }}" alt="cellxgene viewer showing UMAP embedding with cells colored by cluster, gene expression sidebar, and metadata panel" class="screenshot-img">

No installation needed — cellxgene runs as part of the platform and opens directly in your browser.

## QC dashboards

Every pipeline run generates a QC dashboard with key quality metrics:

- Cell counts and viability
- Sequencing read depth and mapping rates
- Gene detection rates
- Doublet scores
- Mitochondrial content

<img src="{{ '/assets/images/screenshot-qc-dashboard.png' | relative_url }}" alt="QC dashboard with summary statistics, violin plots, and quality metric charts" class="screenshot-img">

## Plot archive

All plots generated by pipelines are collected in a searchable archive. You can:

- Browse by experiment, pipeline, or date
- Search by plot name or type
- Download individual plots or bulk export
- View full-resolution versions in the browser

## Sharing results

Results can be shared with anyone on the platform through their role-based access. PIs and lab managers can review results directly without needing to understand the analysis pipeline that produced them.
