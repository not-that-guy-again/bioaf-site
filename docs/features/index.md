bioAF is a complete platform for computational biology at small biotech companies. Here's what's included.

## Core capabilities

- **[Experiment Management](experiments/)** — MINSEQE-compliant metadata, sample tracking, full status lifecycle
- **[Pipeline Engine](pipelines/)** — Run Nextflow and Snakemake workflows with real-time monitoring
- **[Notebook Sessions](notebooks/)** — JupyterHub and RStudio with versioned environments
- **[Results & Visualization](results/)** — cellxgene, QC dashboards, and a searchable plot archive
- **[Data Management](data/)** — File upload, download, and automatic lifecycle management
- **[Cost Center](cost-center/)** — Cloud spending visibility, per-component breakdown, budget alerts
- **[Audit & Compliance](audit/)** — Immutable audit trail with full provenance
- **[Infrastructure Management](infrastructure/)** — Enable and disable platform components from the UI
